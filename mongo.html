MongoDB Asoslari

MongoDB — bu hujjatga asoslangan NoSQL ma’lumotlar bazasidir. Bu darsda siz CRUD (Create, Read, Update, Delete) operatsiyalarini o‘rganasiz.

1. MongoDB o‘rnatish va ulanish

Mongoose orqali ulanish:

const mongoose = require("mongoose");

mongoose.connect("mongodb://127.0.0.1:27017/smartlearn", {
  useNewUrlParser: true,
  useUnifiedTopology: true
});

2. Model yaratish

const User = mongoose.model("User", {
  name: String,
  age: Number
});

3. Ma’lumot qo‘shish (Create)

const newUser = new User({ name: "Ali", age: 25 });
newUser.save();

4. Ma’lumot o‘qish (Read)

User.find().then(data => console.log(data));

5. Ma’lumot yangilash (Update)

User.updateOne({ name: "Ali" }, { age: 30 });

6. Ma’lumot o‘chirish (Delete)

User.deleteOne({ name: "Ali" });

7. Amaliy Mashq

  • Mongoose orqali MongoDB bazasiga ulang
  • Model yarating va CRUD amallarni bajaring
  • Postman orqali test qiling

Agar bularni bajargan bo‘lsangiz — siz MongoDB asoslarini o‘rgandingiz! ✅
