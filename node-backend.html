Node.js Backend Kursi

Loyiha: RESTful API (Express.js)

Bu kursda siz Express.js yordamida quyidagi yo‘nalishlarda REST API yaratishni o‘rganasiz:

  • GET /users — Foydalanuvchilar ro‘yxatini olish
  • POST /users — Yangi foydalanuvchi qo‘shish
  • PUT /users/:id — Ma’lumotni yangilash
  • DELETE /users/:id — Foydalanuvchini o‘chirish
⬅ Kurslar sahifasiga qaytish
