React + API Kursi

Loyiha: Foydalanuvchi ro‘yxatini API orqali yuklash

Bu darsda siz Fetch API yordamida tashqi API'dan ma'lumot olib, uni sahifada chiqarishni o‘rganasiz:

import React, { useEffect, useState } from 'react';

function Users() {
  const [users, setUsers] = useState([]);

  useEffect(() => {
    fetch('https://jsonplaceholder.typicode.com/users')
      .then(res => res.json())
      .then(data => setUsers(data));
  }, []);

  return (
    <div>
      <h3>Foydalanuvchilar</h3>
      <ul>
        {users.map(user => (
          <li key={user.id}>{user.name} ({user.email})</li>
        ))}
      </ul>
    </div>
  );
}
