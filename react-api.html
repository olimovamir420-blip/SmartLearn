React + API

React ilovalari foydalanuvchiga real va dinamik ma’lumotlarni ko‘rsatish uchun API (Application Programming Interface) bilan bog‘lanadi. Ushbu darsda siz REST API’dan ma’lumot olish va uni ekranga chiqarishni o‘rganasiz.

1. Fetch funksiyasi bilan ma’lumot olish

React komponentida API’dan ma’lumot olish uchun useEffect va fetch ishlatiladi:

import { useEffect, useState } from 'react';

function App() {
  const [posts, setPosts] = useState([]);

  useEffect(() => {
    fetch('https://jsonplaceholder.typicode.com/posts')
      .then(res => res.json())
      .then(data => setPosts(data));
  }, []);

  return (
    <div>
      <h2>Postlar ro‘yxati</h2>
      <ul>
        {posts.map(post => (
          <li key={post.id}>{post.title}</li>
        ))}
      </ul>
    </div>
  );
}

3. Yuklanayotgan vaqtni ko‘rsatish

API ma’lumot kelyotganda foydalanuvchiga “Yuklanmoqda...” degan yozuvni ko‘rsatish foydali:

const [loading, setLoading] = useState(true);

useEffect(() => {
  fetch(...).then(res => res.json())
    .then(data => {
      setData(data);
      setLoading(false);
    });
}, []);

return (
  <div>
    {loading ? <p>Yuklanmoqda...</p> : <DataList data={data} />}
  </div>
);

4. Xatolikni tutish (error handling)

Xatolik yuz berganda foydalanuvchiga bildirish uchun try/catch yoki .catch() ishlatiladi:

useEffect(() => {
  fetch('https://api.example.com/data')
    .then(res => {
      if (!res.ok) {
        throw new Error('Tarmoq xatosi');
      }
      return res.json();
    })
    .then(data => setData(data))
    .catch(error => setError(error.message));
}, []);

5. Amaliy loyiha g‘oyasi

API bilan ishlashni mustahkamlash uchun quyidagi loyihani bajaring:

  • Foydalanuvchilar ro‘yxatini API orqali oling
  • Ularning ismlarini ro‘yxatda chiqaring
  • Yuklanayotgan vaqtni va xatolik holatini ko‘rsating

Ushbu amaliyot sizni real React ilova tuzishga tayyorlaydi!
